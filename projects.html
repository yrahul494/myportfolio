Project|01
Project | RAILWAY TRACK FAULT DETECTION USING ARDUINO
  • The Goal of this project is to avoid the Railway Accidents by using "Railway Track Fault Detection Vehicle" .
  • The main components of this project are Arduino UNO, Ultrasonic Sensor, GPS ,Robert Platform .
  • While Moving on the Track if there will be any Cracks or Faluts occur the Vehicle stop there and the location will be shared to predefined numbers using GPS.
  • The Railway Engineer is able to Identify the location of Derailment and take immediate action to Reconstruct the Track.
